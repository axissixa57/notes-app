** etc folder needs to restore configs and other 
** config.json - тут будет храниться порт для сервера, название бд, хост и порт для бд, а также api префикс(адрес по которому будет обращаться для запросов к нашему API)

** package.json:

! react-hot-loader для вебпака, чтобы преобразовать jsx файлы.

! "scripts": {
    "webpack-devserver": "webpack-dev-server --debug --hot --devtool eval-source-map --output-pathinfo --watch --colors --inline --content-base public --port 8090 --host 127.0.0.1"
}

--hot позволяет задействовать плагин Hot Module Replacement, который позволяет производить повторный рендеринг при изменениях в компоненте. 
--open запускает браузер для обращения к приложению.
- webpack-dev-server - запускать код на сервере и смотреть изменения

"react-masonry-component" - библиотека нужна для того чтобы длина всех компонентов заметок была равной(layout ввиде кирпичей). готовый компонент позволяющий использовать функционал (просто иморт) + даёт красивую анимацию при удалении

** start.js: 
этот файл нужен для того чтобы запускался express

** npm:

npm i --save classnames - библиотека позволяет манипулировать в аттрибуте className в react компонентах
